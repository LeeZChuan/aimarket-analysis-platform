## 🎯 使用流程

1. **启动框选**
   - 点击聊天输入框底部的"框选图表区域"按钮 (📦 图标)
   - 图表进入框选模式
   - 顶部显示蓝色提示条
   - 十字光标自动禁用

2. **调整范围**
   - 拖动左右两侧的蓝色圆形手柄
   - 调整选择的时间范围
   - 实时预览选中区域

3. **确认选择**
   - 点击左上角的绿色按钮（✓图标）
   - 弹出确认窗口
   - 显示选择的时间范围和数据统计

4. **完成操作**
   - 在弹窗中点击"确认分析"继续后续操作
   - 或点击"取消"放弃选择
   - 系统自动恢复十字光标

5. **快速取消**
   - 点击框选工具左上角的红色按钮（✕图标）
   - 立即退出框选模式
   - 恢复正常图表状态

## 🔧 技术细节

### 状态流转
```
正常模式 → 点击框选按钮 → 框选模式（禁用十字光标）
  ↓
调整范围 → 点击确认 → 显示弹窗
  ↓
确认/取消 → 恢复正常模式（启用十字光标）
```

### 事件流
```
用户点击绿色按钮
  ↓
overlay.onClick 触发
  ↓
分发 regionSelectionConfirm 自定义事件
  ↓
ChartPanel 监听并处理
  ↓
显示确认弹窗
```

### 坐标计算
- 使用 overlay 的 points 获取选择范围
- 通过时间戳匹配数据点
- 计算选中的 K线 数量
