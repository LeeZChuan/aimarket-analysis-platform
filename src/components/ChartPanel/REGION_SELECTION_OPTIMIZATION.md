# 图表区域选择交互优化

## 功能概述

优化了图表区域选择功能，提供更好的用户体验和视觉反馈。

## 主要改进

### 1. 禁用十字光标
在框选模式下自动禁用图表的十字光标，避免干扰用户的框选操作。

### 2. 可视化提示条
在框选模式启用时，图表顶部显示蓝色提示条，包含：
- 动画脉冲指示器
- 操作说明文字
- 清晰的交互指引

### 3. 增强的框选工具
在框选区域左上角提供两个操作按钮：
- **绿色确认按钮**（带对勾图标）：确认选择并显示详情
- **红色取消按钮**（带X图标）：取消选择并退出框选模式

### 4. 确认弹窗
点击确认按钮后显示专业的弹窗，展示：
- 选择的时间范围（开始时间、结束时间）
- K线数据统计（数据数量）
- 确认和取消操作按钮
- 平滑的动画效果

## 技术实现

### 状态管理 (useChartStore)
```typescript
- isInSelectionMode: boolean  // 框选模式状态
- selectionData: RegionSelectionData | null  // 框选数据
```

### 组件结构
```
ChartPanel
├── RegionSelectionModal  // 确认弹窗
├── 框选模式提示条
└── horizontalRegionSelection overlay  // 自定义覆盖物
```

### 事件流程
1. 用户点击框选按钮 → 进入框选模式
2. 禁用十字光标，显示提示条
3. 用户调整选择范围
4. 点击确认按钮 → 触发 `regionSelectionConfirm` 事件
5. 显示确认弹窗，展示选择数据
6. 用户确认 → 恢复十字光标，清除框选

## 使用方式

1. 点击聊天输入框中的"框选图表区域"按钮
2. 在图表上拖动蓝色手柄调整选择范围
3. 点击左上角绿色按钮确认
4. 在弹窗中查看选择的数据详情
5. 点击"确认分析"进行后续操作

## 优化效果

- ✅ 避免十字光标干扰
- ✅ 清晰的视觉反馈
- ✅ 直观的操作指引
- ✅ 专业的数据展示
- ✅ 流畅的交互体验
